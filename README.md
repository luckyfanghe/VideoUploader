###  Installing and Running:
#

###  1. Server

``` bash
$ pipenv install

$ pipenv shell

$ export FLASK_APP=app/server.py

$ export FLASK_RUN_PORT=80

$ export FLASK_RUN_HOST=0.0.0.0

$ flask run

or

$ flask run --host 0.0.0.0 --port 3000
```

###  2. The App

``` bash
$ yarn or npm install

$ yarn or npm start
```
